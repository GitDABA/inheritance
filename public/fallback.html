Inheritance Distribution App

Welcome to the Inheritance Distribution Application - a fair way to distribute items among siblings.

We're Upgrading

Our application is currently being updated to serve you better. Please check back in a few minutes.

This application helps families distribute inherited items fairly by:

  • Cataloging all items with photos and descriptions
  • Allowing siblings to assign point values based on personal preference
  • Using an algorithm to create the most equitable distribution
  • Providing transparency throughout the process
